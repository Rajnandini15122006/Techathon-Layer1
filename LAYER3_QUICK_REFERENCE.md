# Layer 3: HRVC Risk Engine - Quick Reference Card

## 🎯 The Formula

```
Risk = (Hazard × Exposure × Vulnerability) ÷ Capacity
```

## 📊 Components Breakdown

### Hazard (H) - What's Coming
```
├─ Rainfall Forecast (40%)      → mm of rain expected
├─ River Level (35%)             → meters above normal
└─ Soil Saturation (25%)         → % moisture content
```

### Exposure (E) - What's at Risk
```
├─ Population Density (60%)      → people per km²
└─ Traffic Density (40%)         → vehicles per hour
```

### Vulnerability (V) - How Susceptible
```
├─ Slum Percentage (40%)         → % informal settlements
├─ Elderly Percentage (30%)      → % population over 65
└─ Low Elevation % (30%)         → % area flood-prone
```

### Capacity (C) - Ability to Cope
```
├─ Shelter Count (40%)           → emergency shelters
├─ Hospital Beds (35%)           → medical capacity
└─ Drain Strength (25%)          → infrastructure quality (0-100)
```

## 🚦 Risk Levels

| Score | Level | Color | Action |
|-------|-------|-------|--------|
| 80-100 | 🔴 CRITICAL | Red | Evacuate NOW |
| 60-79 | 🟠 HIGH | Orange | Prepare to evacuate |
| 40-59 | 🟡 MEDIUM | Yellow | Monitor closely |
| 20-39 | 🟢 LOW | Green | Normal watch |
| 0-19 | 🔵 MINIMAL | Blue | No action |

## 💻 Code Example

```python
from app.services.risk_engine import RiskEngine

engine = RiskEngine()

cell_data = {
    'rainfall_mm': 150,
    'river_level_m': 5.2,
    'soil_saturation_pct': 85,
    'population_density': 25000,
    'traffic_density': 500,
    'slum_percentage': 12,
    'elderly_percentage': 15,
    'low_elevation_percentage': 40,
    'shelter_count': 3,
    'hospital_beds': 150,
    'drain_strength': 65
}

result = engine.calculate_risk_score(cell_data)
# Returns: {'risk_score': 75.23, 'risk_level': 'HIGH', 'components': {...}}
```

## 🎯 Ward Priority List

Automatically generated by:
1. Calculate risk for all grid cells
2. Group by ward_id
3. Compute average risk per ward
4. Sort by risk (highest first)
5. Assign priority ranks

**No separate module needed!** It's a dashboard feature.
